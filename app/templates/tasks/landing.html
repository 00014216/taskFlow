A cloud-native project & task management platform.

Django 4.2 Docker PostgreSQL Redis Nginx GitHub Actions CI/CD

Features

User Authentication

Register, login, logout. Session stored in Redis. Login required for all data pages.

3 Models + Relationships

Label, Project, Task. Many-to-One (Task→Project) and Many-to-Many (Task↔Label) relationships.

Full CRUD Operations

Create, Read, Update, and Delete for both Projects and Tasks — with confirmation dialogs.

Redis Caching

Dashboard stats cached for 2 minutes. Cache invalidated on data changes. Sessions in Redis.

Health Check Endpoint

/health/ tests DB and Redis connectivity, returns JSON status for monitoring.

REST API

/api/tasks/ returns paginated JSON. Supports search and ordering. Auth required.
